
# 矩阵旋转问题

> 一般来说，旋转矩阵是将n*n的矩阵旋转90，180，270°。常见的有两种算法。


## 顺时针旋转九十度

<img width="2848" height="1600" alt="Image" src="https://github.com/user-attachments/assets/deaddba3-ce5c-4efe-bbde-0a018b75a349" />

### 变换组合方式
> 操作步骤：先将矩阵进行转置，再将每一行数组水平翻转。

规范可用代码块如下：
```C++
void CW_turn(vector<vector<int>>* mar, int r)
{
    auto& mat = *mar;
    // 转置
    for(int i = 0 ; i < r; i++){
        for(int j = i + 1 ; j < r ; j++){
            swap(mat[i][j] , mat[j][i]);
        }
    }
    // 每行翻转
    for(int i = 0 ; i < r ; i++){
        reverse(mat[i].begin(), mat[i].end());
    }
}
```

### 直接的坐标关系转换

> 顺时针90°坐标变换：(i,j) = (j,n-1-i)

代码如下：
```C++
vector<vector<int>> CW_turn(vector<vector<int>> *martrix ,int L){
    vector<vector<int>> res_mar(L, vector<int>(L));
    for(int i = 0 ; i < L ; i++){
        for(int j = 0 ; j < L ; j++){
            res_mar[j][L - 1 - i] = (*martrix)[i][j];
        }
    }
    return res_mar;
}
```


## 逆时针旋转九十度

<img width="2848" height="1600" alt="Image" src="https://github.com/user-attachments/assets/c135eec7-45bc-4a95-bc3f-dd6507335e18" />

### 变换组合方式
> 操作步骤：先将矩阵每行左右翻转，再进行转置。

代码块如下：
```C++
void CCW_turn(vector<vector<int>>* mar, int r)
{
    auto& mat = *mar;
    for (int i = 0; i < r; i++)
    {
        reverse(mat[i].begin(), mat[i].end());
    }
    for (int i = 0; i < r; i++)
    {
        for (int j = i + 1; j < r; j++)
        {
            swap(mat[i][j], mat[j][i]);
        }
    }
}
```

### 直接的坐标关系转换

> 逆时针90°坐标变换：(i,j) = (n-1-j,i)

代码如下：
```C++
vector<vector<int>> CCW_turn(vector<vector<int>> *martrix ,int L){
    vector<vector<int>> res_mar(L, vector<int>(L));
    for(int i = 0 ; i < L ; i++){
        for(int j = 0 ; j < L ; j++){
            res_mar[L - 1 - j][i] = (*martrix)[i][j];
        }
    }
    return res_mar;
}
```


## 旋转一百八十度

<img width="2848" height="1600" alt="Image" src="https://github.com/user-attachments/assets/f13c3e4a-2713-4582-839e-dd46756c0ca5" />

### 变换组合方式
> 操作步骤：矩阵上下翻转，再每行左右翻转；两种翻转顺序可以互换。

代码块如下：
```C++
void turn_half_round(vector<vector<int>>* mar, int r)
{
    auto& mat = *mar;
    // 上下翻转
    for(int i = 0; i < r / 2; i++){
        swap(mat[i], mat[r - 1 - i]);
    }
    // 每行左右翻转
    for(int i = 0; i < r; i++){
        reverse(mat[i].begin(), mat[i].end());
    }
}
```

### 直接的坐标关系转换

> 旋转180°坐标变换：(i,j) = (n-1-j,n-1-i)

代码如下：
```C++
vector<vector<int>> Turn180(vector<vector<int>> *martrix ,int L){
    vector<vector<int>> res_mar(L, vector<int>(L));
    for(int i = 0 ; i < L ; i++){
        for(int j = 0 ; j < L ; j++){
            res_mar[L - 1 - i][L - 1 - j] = (*martrix)[i][j];
        }
    }
    return res_mar;
}
```
